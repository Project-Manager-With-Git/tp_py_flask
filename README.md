# tp_py_flask-dev

python语言flask项目模板的组件开发分支.python的最低版本要求为3.8

## 组件包括

| 组件名             | 说明                               |
| ------------------ | ---------------------------------- |
| `api`              | RESTful接口模块                    |
| `serv`             | 服务模块                           |
| `sse`              | sse接口模块                        |
| `ws`               | websocket接口模块                  |
| `main.py.jinja`    | 入口文件模版                       |
| `dockerfile.jinja` | 构造docker镜像的dockerfile文件模板 |
