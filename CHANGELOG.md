# cp-0.0.1

提供了基本的组件,包括:

| 组件名             | 说明                               |
| ------------------ | ---------------------------------- |
| `api`              | RESTful接口模块                    |
| `serv`             | 服务模块                           |
| `sse`              | sse接口模块                        |
| `ws`               | websocket接口模块                  |
| `main.py.jinja`    | 入口文件模版                       |
| `dockerfile.jinja` | 构造docker镜像的dockerfile文件模板 |
